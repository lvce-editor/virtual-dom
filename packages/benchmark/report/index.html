Methodology

What is measured

By default, each operation is measured in five fresh browser processes. Every run includes warmup samples followed by measured samples, and the report shows the median of those run medians. Table operations include virtual DOM work, DOM patching, and forced layout. The focused setProps and native DOM creation cases retain detached elements and exclude layout so their targeted cost stays visible.
