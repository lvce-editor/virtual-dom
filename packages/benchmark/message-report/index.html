Methodology

What the byte count means

The benchmark instruments the copied renderer-process bundle and records only messages received from its dedicated Renderer Worker. Chromium's DevTools Protocol returns a JSON-safe copy after unsupported host objects such as MessagePort are removed. The analysis extracts direct and batched Viewlet.setDom, Viewlet.setDom2, and Viewlet.setPatches calls. Size is the sum of each normalized call encoded as compact UTF-8 JSON. It is a stable payload metric, not an estimate of engine-specific retained heap size.
